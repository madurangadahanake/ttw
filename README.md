## 🌌 1. Overview & Vision

**`TTW (Time to Work)`** කියන්නේ නිකම්ම නිකන් Countdown Timer එකක් නෙමෙයි. මේක **Deep Space & Cyberpunk Aesthetic** එකකින් සකසපු, **Mobile Phones**, Tablets සහ PCs සඳහා **100% Mobile-First Animated Dashboard Application** එකක්.

ඔයා වැඩ කරන **Focus Time** 🎯, Break සඳහා ගන්නා **Stop Time** 🛑, සහ Target එකෙන් පස්සේ සටහන් වන **Overtime** 💖 ඉතා පැහැදිලි Neon Visual Icons & Live Visual Animations මගින් realtime සජීවීව පෙන්නුම් කරයි.
