## 🌌 Overview

**TTW (Time to Work)** කියන්නේ **Deep Space & Cyberpunk Aesthetic** එකකින් යුතු, ඕනෑම **Smart Phone**, Tablet හෝ Computer එකකින් ඉතා පහසුවෙන් භාවිතා කළ හැකි **Responsive Focus & Productivity Application** එකක්.

ඔයා වැඩ කරන **Focus Time**, Breaks සඳහා ගන්නා **Stop Time** සහ Target එක අවසන් වූ පසු නොනැවතී සටහන් වන **Overtime** එක Mobile Screen එකක වුවද පැහැදිලිව, Animations සහිතව Track කිරීමට මෙය උපකාරී වේ.

## 📊 Status Panel Metrics

Mobile View එකෙත් පැහැදිලිව පෙනෙන පරිදි Status metrics පෙළගස්වා ඇත:

| Metric Label | Live Functionality |
| :--- | :--- |
| 🎯 **Target Time** | ආරම්භයේදී Set කළ මුළු ඉලක්කගත කාලය. |
| ⏱️ **Total Elapsed** | ගතවූ සමස්ත කාලය (Countdown + Overtime). |
| 🛑 **Total Stopped** | වැඩ අතරතුර Stop කර තිබූ මුළු කාලය (Break Time Tracker). |
| 💖 **Overtime (අමතර)** | Target එක ඉක්මවා වැඩ කළ අමතර කාලය (Neon Pink Pulse UI). |

## 💾 Local Storage Sync

Browser එක Closes කළද / Mobile Screen එක Lock කළද දත්ත ආරක්ෂිතව පවතී:

```json
{
  "ttw_initial": "Initial target duration (seconds)",
  "ttw_current": "Current remaining countdown (seconds)",
  "ttw_stop_seconds": "Total break/stopped time (seconds)",
  "ttw_is_stopped": "Stop timer state boolean",
  "ttw_is_overtime": "Overtime active state boolean",
  "ttw_overtime_seconds": "Active overtime duration (seconds)"
}
```

## 🚀 Quick Start Guide

1. `index.html` ෆයිල් එක Download කරගන්න.
2. Phone එකෙන් හෝ PC එකෙන් **Chrome / Safari / Firefox** ඕනෑම Browser එකකින් Open කරගන්න.
3. Target කාලය ඇතුළත් කර **Start Timer** ලබා දී ඔබේ Focus Session එක පටන් ගන්න!
